# Project Handover: Aasta Ayur Ladies Beauty Lounge

## Project Type

Static multi-page website for a ladies beauty lounge and wellness destination.

Pages included:

- `index.html`: Homepage and brand overview
- `services.html`: Hair, skin, nails, threading, waxing, and beauty services
- `moroccan-bath.html`: Signature Moroccan Bath ritual page
- `bridal.html`: Bridal, event beauty, and package concepts
- `booking.html`: WhatsApp-first booking paths
- `visit.html`: Location, contact, hours, and FAQ

## What Is Production-Ready

- Mobile-first responsive layout
- Multi-page navigation
- Static hosting configuration for Render, Netlify, and Vercel
- Original generated visual assets stored in `assets/`
- Service-specific WhatsApp CTA messages
- Client-facing content structure
- Deployment instructions

## What Must Be Confirmed With The Business

- Verified WhatsApp number
- Phone number, if different from WhatsApp
- Instagram URL
- Email address, if used
- Exact building address
- Opening hours
- Google Maps link or embed
- Current service menu
- Package names and pricing
- Whether Moroccan Bath copy should use “authentic,” “traditional,” or softer wording
- Whether the private demo concept note should stay or be removed

## Content Editing Guide

Most content lives directly in the page HTML files. The main repeated edits are:

- Navigation: update in each HTML page header and footer
- WhatsApp number: search for `971000000000`
- Instagram placeholder: search for `https://www.instagram.com/`
- Email placeholder: search for `hello@example.com`
- Location placeholder: search for `Exact building`
