# Aasta Ayur Ladies Beauty Lounge Demo

Private demo concept for Aasta Ayur Ladies Beauty Lounge in Al Zahiyah / Tourist Club Area, Abu Dhabi.

This is a static HTML/CSS/JS website concept designed around:

- Ayurvedic-inspired beauty care
- Moroccan Bath ritual positioning
- Ladies-only sanctuary atmosphere
- Mobile-first WhatsApp booking
- Service-specific inquiry flows

## Preview Locally

```bash
python3 -m http.server 8123
```

Then open `http://localhost:8123/`.

## Deployment

This repo is deployment-ready for static hosting:

- Render: `render.yaml`
- Netlify: `netlify.toml`
- Vercel: `vercel.json`

See `DEPLOY.md` before publishing.
